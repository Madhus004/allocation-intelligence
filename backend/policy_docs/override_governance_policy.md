# Override Governance Policy

## Purpose
Define when the allocation review agent is allowed to recommend overriding the OMS-selected allocation.

## Policy Rules
- The agent may recommend an override only when the score improvement over the OMS-selected option is material.
- For prototype purposes, a material improvement means score gap >= 10.
- If the recommended option increases fulfillment cost but materially reduces SLA risk, store stockout risk, or replenishment risk, override may still be justified.
- If the recommendation would increase split shipments, it must show a clear improvement in business value or service confidence.

## Guidance
- Keep the OMS-selected option when the improvement is marginal.
- Prefer the scorer-recommended option when it provides meaningful inventory protection and still meets promise.
